== Description ==

**Auto-Legal & Compliance Assistant** by Anthony Acosta helps website owners stay compliant with zero hassle.

**Free Features**
* Instant Privacy Policy Generator
* Terms of Service Generator
* Responsive Cookie Consent Banner
* Clean shortcode display

**Pro Features**
* Auto-updated policies when laws change
* Geo-aware cookie banners (EU, CA, US states)
* Monthly compliance email reminders
* Sync across multiple sites
* White-label compliance badges

**Important**: Templates only. Always consult a licensed attorney.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin
3. Go to your WordPress Dashboard – the new widget appears automatically

== Shortcodes ==

* `[alc_privacy_policy]`
* `[alc_terms_of_service]`
* `[alc_cookie_banner]`
* `[alc_compliance_badge]` (Pro)
